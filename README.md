## 📋 Sobre o Projeto

O **Festivite** é uma aplicação web moderna que permite criar convites digitais personalizados para diversos tipos de eventos. Com uma interface intuitiva e design responsivo, você pode personalizar cores, temas, estilos e muito mais para criar o convite perfeito.

### ✨ Funcionalidades

- 🎨 **Personalização Completa**: Escolha entre 11 cores diferentes e 12 temas de eventos
- 📱 **Design Responsivo**: Interface adaptável para desktop e mobile
- 🌙 **Modo Escuro/Claro**: Toggle para alternar entre estilos
- 📸 **Upload de Imagens**: Adicione sua própria foto de capa
- 🎭 **Temas Diversos**: Aniversário, casamento, formatura, carnaval, natal e muito mais
- ⚡ **Animações Suaves**: Efeitos visuais modernos com animações RGB
- 📝 **Formulário Inteligente**: Validação em tempo real e campos obrigatórios
- 👥 **Dados de Contato**: Coleta informações para confirmação de presença

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada com variáveis CSS

## 📦 Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/seu-usuario/festivite.git
   cd festivite
   ```

2. **Abra o projeto**
   ```bash
   # Abra o arquivo index.html no seu navegador
   open index.html
   # ou
   # Use um servidor local
   python -m http.server 8000
   ```

3. **Acesse no navegador**
   ```
   http://localhost:8000
   ```

## 🎯 Como Usar

1. **Preencha os dados do evento**
   - Título do evento
   - Data e horário de início e fim
   - Tipo (presencial ou online)
   - Local ou link
   - Descrição detalhada

2. **Personalize o visual**
   - Escolha uma cor principal
   - Selecione o tema do evento
   - Ative/desative modo escuro
   - Faça upload de uma foto de capa

3. **Adicione seus dados de contato**
   - Nome completo
   - E-mail
   - Telefone
   - Aceite os termos e condições

4. **Gere seu convite**
   - Clique em "Gerar Convite"
   - Visualize o preview ao lado
   - Compartilhe com seus convidados!
